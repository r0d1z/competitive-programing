# Competitive Programming Solutions

This repository contains a collection of solutions to various competitive programming problems, primarily implemented in C++. The solutions focus on algorithmic efficiency and covers a range of topics including dynamic programming, greedy algorithms, and data structures.

## Structure

The project is organized by problem, with each C++ file corresponding to a specific challenge:

- apartments.cpp
- array_desc.cpp
- bit_strings.cpp
- book_shop.cpp
- coin_combinations.cpp
- coinII.cpp
- concert_tickets.cpp
- dice_combinations.cpp
- distinct_number.cpp
- minimizin_coins.cpp
- movie_festival.cpp
- palindrome_reorder.cpp
- permutations.cpp
- removing_digits.cpp
- two_set.cpp

## Build and Run

To compile any of the solutions, use a C++ compiler such as g++. For example:

    g++ -O3 -o solution problem_name.cpp
    ./solution

## Usage

These solutions are intended for educational purposes and as a reference for common algorithmic patterns used in competitive programming.
